# Merge Sort Pattern

## 📌 What is Merge Sort?

Merge Sort is a Divide and Conquer algorithm.

It works by:

1. Dividing the array into two halves.
2. Solving each half recursively.
3. Merging the sorted halves.

---

## 🧠 When to Think of Merge Sort

Use Merge Sort when the problem asks for:

- Counting inversions
- Reverse pairs
- Counting smaller elements after self
- Stable sorting
- Efficient sorting while collecting extra information

---

## 🚀 General Algorithm

1. Divide the array.
2. Solve recursively.
3. Count required information.
4. Merge the sorted halves.

---

## 📊 Complexity

Time: O(n log n)

Space: O(n)
