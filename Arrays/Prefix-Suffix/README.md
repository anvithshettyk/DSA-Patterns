# Prefix-Suffix Traversal

## 📌 What is the Prefix-Suffix Pattern?

The Prefix-Suffix pattern processes an array from both directions to capture information that may be missed by a single traversal.

It is useful when the result depends on elements before and after a position.

---

## 🧠 When to Use

- Product-based problems
- Prefix and suffix computations
- Bidirectional array traversal

---

## 📊 Complexity

Time: O(n)

Space: O(1)
