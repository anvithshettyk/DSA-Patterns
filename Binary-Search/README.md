# 🔍 Binary Search

Binary Search is an efficient searching algorithm that works on **sorted arrays** by repeatedly dividing the search space into half.

---

## 🧠 When to Use

- The array is **sorted**.
- You need to search for an element efficiently.
- The problem asks for **O(log n)** time complexity.
- Finding the first/last occurrence of an element.
- Searching in rotated sorted arrays.
- Finding minimum/maximum in sorted structures.

---

## ⚡ Complexity

| Operation | Time | Space |
|-----------|------|-------|
| Search | O(log n) | O(1) |
